# Мануфактура «Матрёшка» — интернет-магазин

Полноценный интернет-магазин товаров ручной работы из Читы (Забайкальский край).
Node.js + Express + встроенный SQLite (`node:sqlite`) + ванильный SPA-фронтенд.

## Запуск

```bash
npm install      # установка зависимостей
npm start        # запуск сервера
```

Откройте **http://localhost:3000**

База данных `db.sqlite` создаётся и наполняется тестовыми данными автоматически при первом запуске.

## Возможности

- **Каталог** с фильтрами (категория, материал, цена, наличие) и поиском с автодополнением.
- **Карточка товара**: галерея, «С этим покупают», отзывы.
- **Корзина и оформление**: самовывоз / Почта России / СДЭК; оплата Мир / СБП / наличными; промокод, подарочная упаковка, комментарий.
- **Авторизация** (bcrypt + JWT, сессия в localStorage). Личный кабинет: заказы, избранное, адреса.
- **Меню «⋯»** (пароль `7316`):
  - *Редактировать каталог* — добавление/редактирование/удаление товаров с загрузкой фото.
  - *Панель модерации* — все заказы в реальном времени (polling каждые 10 с), смена статусов.
- Страницы: Главная, О нас, Контакты (карта Яндекс), Блог, Акции, Оптовикам (B2B), Возврат и гарантия.
- Адаптивная вёрстка (mobile-first), SEO-метатеги, ЧПУ-ссылки, ленивая загрузка изображений.

## Структура

```
server.js       — Express-сервер и REST API
database.js     — схема SQLite и наполнение данными
public/
  index.html    — оболочка SPA
  css/style.css — оформление (тёплая народная палитра)
  js/app.js      — клиентская логика (роутер, корзина, авторизация, админ-панели)
db.sqlite       — файл базы данных
```

## API (основное)

- `POST /api/auth/register` · `POST /api/auth/login` · `GET /api/auth/me`
- `GET /api/products` · `GET /api/products/:slug` · `GET /api/products/facets` · `GET /api/products/suggest`
- `POST /api/orders`
- Админ (заголовок `X-Admin-Password: 7316`): `POST/PUT/DELETE /api/admin/products`, `GET /api/admin/orders`, `PUT /api/admin/orders/:id/status`
